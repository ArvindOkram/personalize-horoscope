### Step to setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/ArvindOkram/personalize-horoscope.git
   cd personalize-horoscope

   ```

2. **Install dependencies**

   ```bash
   npm install

   ```

3. **Run Mongo Locally**

   ```bash
   mongod
   in seperate terminal

   ```

4. **Start the serve**
   ```bash
   npm run start:dev
   ```
